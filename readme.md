# The Wild Oasis Website

## Description

Amazing website allows you to book wild cabin in the nature
with simple way , best experience and kindfull service.

## Techniques Used

- HTML
- CSS
- Javascript
- React
- Vite
- supabase

  ## Sections

  - Home
  - Bookings
  - Cabins
  - Users
  - Settings
  - Updating Account

    ## How to run
 
    ### Firstly
 
    Open the Supabase server

    ### Secondly
    #### From the command line run

    - npm i
    - npm run dev
